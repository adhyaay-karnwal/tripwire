# Contribution rules for bountydotnew/tripwire

These rules are enforced automatically by [Tripwire](https://tripwire.sh). Contributions that violate them may be closed or held for review.

## What we check

- **Account age** — accounts younger than 14 days are logged.
- **Merged PR history** — contributors with fewer than 5 merged PRs (anywhere on GitHub) are logged.
- **Public repo activity** — contributors with fewer than 3 public repos are logged.
- **Profile README** — contributors without a GitHub profile README are logged.
- **PR rate limit** — more than 5 PRs per day from a single contributor are logged.
- **Cryptocurrency addresses** — content containing wallet addresses (BTC, ETH, SOL, XMR, DASH) is logged.
